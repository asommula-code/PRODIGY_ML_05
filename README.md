# PRODIGY_ML_05
## Task 5
Develop a model that can accurately recognize food items from images and estimate their calorie content, helping users monitor dietary intake and make informed food choices.

## Dataset
- Source: Food-101 Dataset
- Link: https://www.kaggle.com/datasets/dansbecker/food-101
- Contains images from 101 food categories.

## Outcome
Created a food recognition system that identifies food items and provides calorie estimates for dietary tracking.
